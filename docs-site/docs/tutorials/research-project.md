# Research a Topic

Use Research Mode for deep-dive analysis with web search and multi-step reasoning.

## What You'll Learn

- How to activate Research Mode
- How the system gathers and synthesizes information
- How to refine research results

## Step 1: Activate Research Mode

In the Hive UI, either:

- Toggle the **Research Mode** switch, or
- Prefix your message with a research-style query

## Step 2: Ask a Research Question

> Research the latest developments in quantum computing error correction

The system will:

1. **Break down** your question into sub-queries
2. **Search** the web using the Search Tool
3. **Synthesize** findings from multiple sources
4. **Cite** sources in the response

## Step 3: Review Results

The response includes:

- A structured summary
- Key findings organized by theme
- Source citations with links
- Areas of uncertainty or debate

## Step 4: Go Deeper

Follow up with targeted questions:

> Tell me more about surface codes vs. color codes

The agent maintains context from the initial research and provides more detailed analysis.

## Step 5: Export

> Summarize your findings as a markdown report

The agent generates a structured document you can copy or save.

## Tips

- Be specific: "quantum computing error correction in 2024" vs. "quantum computing"
- Ask for comparisons: "Compare X vs Y"
- Request specific formats: "Give me a table comparing approaches"
