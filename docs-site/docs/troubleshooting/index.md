# Troubleshooting

Symptom-based guides for diagnosing and resolving issues. Each page covers a specific component.

| Component | Common Issues |
|-----------|--------------|
| [Ollama](ollama.md) | Model load failures, OOM, slow inference |
| [ComfyUI](comfyui.md) | Workflow errors, VRAM conflicts |
| [SPIRE](spire.md) | Attestation failures, expired SVIDs |
| [Network](network.md) | Connectivity, DNS, Traefik routing |
| [GPU](gpu.md) | Driver issues, VRAM, thermal throttling |
| [Agent Runtime](agent-runtime.md) | Crashes, timeouts, intent routing |
| [Langfuse](langfuse.md) | Tracing failures, database connection |
| [Docker](docker.md) | Container failures, disk space, networking |
| [Voice](voice.md) | STT/TTS failures, audio issues |

!!! tip
    For quick answers, try the [Troubleshooting FAQ](../faq/troubleshooting-faq.md) first.
