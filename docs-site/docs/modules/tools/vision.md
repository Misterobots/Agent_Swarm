# Vision Tool

Image analysis using a Vision Language Model.

## Functions

| Function | Description |
|----------|-------------|
| `analyze_image(image_path, question)` | Analyze an image with a question |

## Model

| Property | Value |
|----------|-------|
| Model | Moondream (moondream:latest) |
| VRAM | ~2 GB |
| Speed | ~80 tokens/sec |

## Use Cases

- Describe image contents
- Answer questions about images
- Extract text from screenshots
- Identify objects, people, scenes

## Allowed Intents

`VISION`
