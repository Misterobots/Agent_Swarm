# Home Assistant Add-ons

Native Home Assistant add-on scaffolds that should be installed on the Home Assistant box rather than on Hive or the Turing.

## Add-ons

| Add-on | Purpose | Folder |
|--------|---------|--------|
| Mission Control | Cooperative smart-home challenge game | `mission-control/` |

## Install target

Copy each add-on folder into the Home Assistant local add-ons directory:

- `/addons/local/mission_control`

After copying, open the Home Assistant Add-on Store and run `Check for updates`.
